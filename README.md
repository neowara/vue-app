# **TVMaze Vue App**
This project was built using Vue 2.6 and TypeScript 4.5 
Uses [Link](https://www.tvmaze.com/api) as API

---
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
